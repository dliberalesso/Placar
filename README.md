# Placar Copa URI de Robótica

Trabalho desenvolvido para a disciplina de Linguagem III, do curso de Ciência da Computação da URI Santo Ângelo.
Tem por objeto a criação de um placar para a Copa de Robótica da URI, utilizando a linguagem Java.

A Documentação pode ser encontrada em [http://dliberalesso.me/Placar/](http://dliberalesso.me/Placar/).
